Como funciona a compra e venda de moedas estrangeiras?

A compra e venda de moedas estrangeiras (também conhecido como câmbio) serve para poder trocar moedas nacionais por moedas estrangeiras ao fazer uma viagem ao exterior de um país, utilizando a moeda do país destinatário.

Para cambiar moedas, é necessário ir para agencias que trabalham com tal ato, que geralmente são encontradas em aeroportos. Cambiar não é uma tarefa dificil, é preciso apenas possuir um valor qualquer da moeda nacional, apresentar documentos caso seja necessário, pagar uma taxa de câmbio e realizar a troca, o mesmo vale para o processo de conversão para a moeda nacional.

Exemplo de cambio

Para cambiar um real para dólar, é necessário possuir mais do que 5 reais aproximadamente (preço do dólar, em reais) e com isso, é feito um cálculo para saber o valor recebido em dólar

Suponha que um cliente queira viajar para os Estados Unidos, querendo levar consigo 10.000 dólares, para isso, o mesmo precisa se apresentar na agência de câmbio, mostrar os documentos necessários, assim como pagar a taxa e apresentar seu valor em reais (que no caso seria 50.000), com isso, o cáculo de 50.000/5 é feito para que seja recebido os 10.000 dólares convertidos.
